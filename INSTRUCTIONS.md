# Workshop instructions

1. Run the script `setup.sh` from the project folder root, or manually add the following to `.git/info/exclude`:
    ```
    .github/copilot-instructions.md
    .github/git-commit-instructions.md
    .github/instructions/**.md
    ```
2. Copy the contents of https://github.com/infinum/ai-rules/tree/feature/java-ai-workshop-01/backend/java/ai-workshop
(`.github` folder) to your repository.
3. Make sure that the Copilot extension is installed in your IDEA by following the instructions located here: https://docs.github.com/en/copilot/how-tos/get-code-suggestions/get-ide-code-suggestions?tool=jetbrains.
4. Enjoy the workshop!
